- This project using flyway to create data and tables in database
- You must create a database name: simplehr (using MySQL)
- Then, open terminal in the root folder of your project, run the following command:
  mvn fly:migrate
- After that, all data will be inserted into your database.

Các topic trong project này:
- flyway
- hibernate (hibernate query, states,...)
- Entity manager vs hibernate session
- pagination in hibernate
- Tạo ra hệ thống Bảng từ các class Entity trong Hibernate
